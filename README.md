# Onegin
Education MIPT task to write a cli program that sorts line in given text different ways

## Building
```bash
$ mkdir build
$ cd build
$ cmake -DCMAKE_BUILD_TYPE=Release ..
$ make
```
## Usage
```bash
$ ./gqsort -f FILENAME
```
