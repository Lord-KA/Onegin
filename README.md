# Onegin
MIPT task to write a cli program that sorts line in a large text.

## Building
```bash
$ mkdir build
$ cd build
$ cmake -DCMAKE_BUILD_TYPE=Release ..
$ make
```
## Usage
```bash
$ ./gqsort -f FILENAME
```
